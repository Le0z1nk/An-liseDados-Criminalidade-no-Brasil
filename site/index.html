Dashboard Executivo de Criminalidade no Brasil

Projeto de análise e visualização de dados com Python, Pandas e Streamlit

Sobre o Projeto

Este projeto apresenta uma análise executiva de criminalidade no Brasil, considerando tipo de crime, ocorrencias, vitimas, índice de violência e nível de risco

A proposta é demonstrar um fluxo completo de projeto analítico: preparação dos dados, análise exploratória, criação de indicadores, visualizações, dashboard interativo e publicação em ambiente web

Perguntas de Negócio

  • Quais cidades apresentam maior número de ocorrências?
  • Quais tipos de crime são mais frequentes?
  • Existem horários mais críticos?
  • Há crescimento ou redução da criminalidade ao longo do tempo?
  • Existem diferenças regionais relevantes?
  • Há relação entre renda e criminalidade
  • Quais regiões apresentam maior índice de violência?
